== Description ==

Description: Chromium is an Auto Parts Shop WordPress Themes. Chromium is Responsive and highly customizable. Chromium theme will be a good fit for a wedding, fashion or personal blog, website. Chromium theme fully supports WooCommerce plugin so you can easily turn your blog into an e-commerce website. Chromium theme supports Elementor so you can easily create static pages with no coding skills. Chromium Theme Demo can be found here: http://chromium.themes.zone

2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
Chromium theme supports Elementor
Chromium theme supports WooCommerce
Chromium theme supports Revolution Slider
